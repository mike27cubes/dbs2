INSTALLATION
============

## Installing via Composer

Add the following repository

```
{
    "type": "vcs",
    "url": "https://github.com/mike27cubes/dbs2"
}
```

Add the following dependency requirement

```
"27cubes/dbsmart2": "dev-master"
```
